# Sentences
Классификация предложений на автоматически сгенерированные и естественные
В файле text.csv представлена выборка предложений, половина из которых является настоящими предложениями (с нормализованными словами) из книги Л. Улицкой, а половина - сгенерированные предложения из слов из этой же книги. В файле classification.ipynb построен классификатор, отличающий настоящие предложения от сгенерированных (кроссвалидация на представленной выборке).
